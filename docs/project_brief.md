# 프로젝트 브리프 — Coffee & F&B Market Intelligence

> **이 문서의 용도**: 다른 대화창(특히 도메인 학습용)에 붙여넣어 맥락을 빠르게 공유하기 위한 요약본.
> 프로젝트가 처음인 사람(또는 AI)이 이것만 읽어도 "무엇을, 왜, 어디까지 했고, 다음에 뭘 하는지"를 알 수 있도록 작성.

## 1. 프로젝트 개요

- **무엇**: 한국 커피 시장에 대한 **월간 시장 인텔리전스 리포트**를 만드는 포트폴리오 프로젝트.
- **왜**: 작업자(금융 IT BA 5년차, 전직 Java 개발자)가 **AI 활용력 + F&B/커피 도메인 지식**을 결합한
  *AI-enabled Domain BA/Analyst* 로 커리어를 확장하기 위한 결과물.
- **핵심 산출물**: `reports/monthly_coffee_market_brief_YYYY_MM.md`
- **저장소**: github.com/jaeahn91/coffee-fnb-market-intelligence
- **작업 방식**: `docs/project_log.md` 에 **Day 단위 로그**로 기록, 매 세션 작은 결과물 하나씩, 단계별 커밋.

### 리포트 표준 구조 (7 섹션)
1. Executive Summary
2. **Green Bean Import Trend** ← 데이터 확보 완료
3. **International Coffee Price Trend** ← 데이터 확보 완료
4. FX Impact
5. Origin and Supply Risk
6. Implications for Korean F&B Market
7. Next Month Watchpoints

### 확보한 데이터 자산
- `data/raw/korea_green_bean_imports_hs090111_2024_2026.csv` — 국가별 월별 생두 수입(중량 kg, 금액 USD)
- `data/raw/coffee_prices_fred_2024_2026.csv` — 국제 아라비카/로부스타 월별 가격 (센트/lb + USD/kg 환산)
- `data/processed/monthly_import_summary.csv` — 월별 총물량·총금액·평균단가·국가수
- `data/processed/origin_summary.csv` — 원산지별 누적 물량·점유율
- `data/processed/price_vs_import_unitprice.csv` — 수입단가 vs 국제시세 비교

### 데이터 소스 (확정)
- **수입**: 관세청 OpenAPI(data.go.kr) — `getNitemtradeList`, XML, `impWgt`(kg)/`impDlr`(USD). API 키는 `.env`(비공개).
- **가격**: FRED(IMF Primary Commodity Prices) — `PCOFFOTMUSDM`(아라비카), `PCOFFROBUSDM`(로부스타). 무료, 키 불필요.
- (예정) ICO I-CIP = 가격 권위 소스로 나중에 교차검증, 한국은행/FRED = 환율.

## 3. 지금까지 나온 핵심 발견 (2가지)

1. **수입 단가가 약 2배** ($3.9/kg → ~$7/kg, 2024-01 → 2026 초). **물량은 평탄**(월 12~16천 톤)
   → 수입 *금액* 증가는 물량이 아니라 **가격이 견인**.
2. 그 단가 상승은 **국제 아라비카 시세와 상관 0.87** (로부스타와는 0.26)
   → 한국 비용 상승은 국내 요인이 아니라 **세계 아라비카 시세 급등이라는 외생적 비용 충격**으로 설명됨.
3. (참고) **원산지 집중**: 상위 4개국(브라질 33.9% · 베트남 16.7% · 콜롬비아 16.5% · 에티오피아 12.2%) = 물량의 약 **79%**.

### 주제 1에서 남긴 "열린 질문"(다음에 답해볼 것)
- 한국 수입 생두의 **아라비카:로부스타 비율**은? (HS 090111은 종 구분 안 함 → 별도 데이터 필요)
- "수입 단계 가격"과 "소비자 가격" 사이의 **비용 레이어**는 몇 개인가? (관세·운송·로스팅·임대·마진)
- 생두는 **재고 비축 가능** → 이번 달 수입량 ≠ 이번 달 소비량 (재고 효과)
