## v0 — "한국 커피 수입 비용이 왜 오르나?" *(FROZEN)*

- **발행호:** 2026년 5월호 · **data-cut:** 수입 2026-04 / 국제가격 2026-03 / 환율 2026-05
- **답한 질문:** 한국 생두 수입 비용 급등의 원인은?
- **결론:**
  - 비용 증가의 주동인은 **물량이 아니라 가격** (단가 $3.9 → ~$7/kg, 물량은 평탄).
  - 단가 상승은 **세계 아라비카 시세라는 외생 충격** (레벨 상관 0.87, 단 차분 0.08 → 동행이지 단기 견인 아님).
  - 원화 원가 ~2배 (5,150 → 11,030원/kg) — 기여도 **가격 ~85% / 환율 ~15%**; 환율이 시세 하락분을 상쇄해 고점 고착.
  - 원산지 상위 4개국 ~79% 집중; 에티오피아 4월 단가 프리미엄 +$1.07 반등(공급 타이트 신호).
- **남긴 것 (→ v1):** §6 국내 전가는 *가설*; §7 트리거 3개는 다음 데이터로 검증 예정.
- **재현 가드:** `scripts/check_figures.py`의 골든 13개로 위 숫자 고정 (드리프트 시 빨간불).
