## v0 — "한국 커피 수입 비용이 왜 오르나?" *(FROZEN)*

- **발행호:** 2026년 5월호 · **data-cut:** 수입 2026-04 / 국제가격 2026-03 / 환율 2026-05
- **답한 질문:** 한국 생두 수입 비용 급등의 원인은?
- **결론:**
  - 비용 증가의 주동인은 **물량이 아니라 가격** (단가 $3.9 → ~$7/kg, 물량은 평탄).
  - 단가 상승은 **세계 아라비카 시세라는 외생 충격** (레벨 상관 0.87, 단 차분 0.08 → 동행이지 단기 견인 아님).
  - 원화 원가 ~2배 (5,150 → 11,030원/kg) — 기여도 **가격 ~85% / 환율 ~15%**; 환율이 시세 하락분을 상쇄해 고점 고착.
  - 원산지 상위 4개국 ~79% 집중; 에티오피아 4월 단가 프리미엄 +$1.07 반등(공급 타이트 신호).
- **남긴 것 (→ v1):** §6 국내 전가는 *가설*; §7 트리거 3개는 다음 데이터로 검증 예정.
- **재현 가드:** `scripts/check_figures.py`의 골든 13개로 위 숫자 고정 (드리프트 시 빨간불).

## v1 — "비용 충격이 국내로 실제 전가됐나? + 공급 신호는 진짜였나?" *(발행 완료 — 트리거 검증호)*

- **발행호:** 2026년 6월호 (`reports/monthly_coffee_market_brief_2026_06.md`) · **data-cut:** 수입·국제가격 2026-05 / 환율 2026-06(부분)
- **답한 질문:** ① 상류 원가 충격이 국내 소매가로 전가됐나(얼마나)? ② 4월 공급 신호가 일시적이었나, 지속인가?
- **결론:**
  - **§7 트리거 3개 모두 v0 가설과 같은 방향으로 확정(반증되지 않음):** A-1 ET 프리미엄 **+$1.03/kg**(2개월 연속 ≥ +$0.8 → 공급 타이트 지속) / A-2 FX 상쇄 **지속**(국제 아라비카 정점 대비 −20.1%인데 환율 +7.4%가 상쇄해 원화 원가 −4.8%, 여전히 10,542원 > 10,500 임계) / A-3 물량 **11,411톤 < 13,000**(2026 평균 11,891, 4월 반등은 고립된 튐 → 둔화 강화).
  - **§6 국내 전가 첫 검증(CPI):** 상류 원화 원가 **+114%** vs 외식 커피 CPI **+4.4%**(2026-06까지 동일) → *"생두값 2배 ≠ 판매가 2배"* 데이터로 확인. +4.4%는 v0 §6.1 잔당 산식이 예측한 폭(+2~8%) 안에 정량적으로 부합. 감쇠 사다리: 원가 +114% → 원두 소매 +16% → 외식 한 잔 +4%.
- **남긴 것 (→ v2 후보):** 가격대별 전가 **비대칭**(저가 vs 프리미엄)은 집계 CPI로 분해 불가 → 프랜차이즈 구간별 실판매가가 최대 공백(6월호 §5 C-1 신규 트리거). (선택) ICO I-CIP·한국은행 고시환율 권위 소스 교차검증.
- **재현·발행 규율:** 이 호는 **린 발행** — processed/notebook 재생성 없이 2026-05 수치를 raw에서 직접 산출해 판정표에 실었다. v0 골든 13개는 이 호에서 변동 없음(골든 가드가 `data_cut` 윈도우로 고정돼, 새 달 추가가 과거 수치를 바꾸지 않음을 확인). v0 본문은 frozen 유지. processed/골든 재캡처는 차트를 2026-05로 연장하는 시점(구성효과 A-4 재계산 포함)으로 이월.
